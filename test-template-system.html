🚀 Sistema de Template Variables - KenzySites

Sistema desenvolvido para permitir que agentes automatizados alterem conteúdo de landing pages via API.

✅ Implementação Completa: Sistema de template variables totalmente funcional

✅ Parser de placeholders dinâmicos ({{VARIABLE_NAME}})
✅ Sistema de substituição de variáveis

✅ Conversão de conteúdo para template variables

4. Template Dinâmico (templates/page-kenzysites-acf.php)

✅ Renderização dinâmica de variáveis
✅ Suporte a templates médicos
✅ Integração com WhatsApp

🔗 Endpoints da API Disponíveis

GET /wp-json/kenzysites/v1/templates/types → Lista todos os tipos de template disponíveis
GET /wp-json/kenzysites/v1/templates/{id}/variables → Obtém todas as variáveis de um template específico
POST /wp-json/kenzysites/v1/templates/{id}/update → Atualiza variáveis de um template (requer API key)
POST /wp-json/kenzysites/v1/templates/{id}/preview → Gera preview do template com novas variáveis
POST /wp-json/kenzysites/v1/templates/{id}/upload-image → Upload e substituição de imagens

💡 Exemplo de Uso para Agentes

Template Original da Dra. Mariana:

Dr. {{NOME_MEDICO}} - Especialista em {{ESPECIALIDADE}}
📞 {{TELEFONE}} | 📧 {{EMAIL}}
📍 {{ENDERECO}} - {{CIDADE}}
🏥 {{CONSULTORIO}}

Comando do Agente via API:

POST /wp-json/kenzysites/v1/templates/123/update
Header: X-KenzySites-API-Key: sua_api_key

{
  "variables": {
    "NOME_MEDICO": "Dr. João Silva",
    "ESPECIALIDADE": "Cardiologia",
    "TELEFONE": "(11) 99999-9999",
    "EMAIL": "joao@cardio.com",
    "ENDERECO": "Rua das Flores, 123",
    "CIDADE": "São Paulo",
    "CONSULTORIO": "Clínica CardioVida"
  }
}

Resultado Automático:

Dr. João Silva - Especialista em Cardiologia
📞 (11) 99999-9999 | 📧 joao@cardio.com
📍 Rua das Flores, 123 - São Paulo
🏥 Clínica CardioVida

🎯 Benefícios para Agentes Automatizados

  • ✅ Simplicidade: Apenas uma chamada API para atualizar todo o conteúdo
  • ✅ Performance: Sem dependência do ACF, direto no banco
  • ✅ Flexibilidade: Qualquer tipo de conteúdo pode ser variável
  • ✅ Escalabilidade: Suporta múltiplos templates simultaneamente
  • ✅ Monitoramento: Logs completos de todas as alterações

📈 Comparação: ACF vs Template Variables

Aspecto ACF (Anterior) Template Variables (Novo)
Acesso por API ❌ Complexo, múltiplos endpoints ✅ Simples, um endpoint
Performance ❌ Múltiplas consultas no banco ✅ Uma consulta, dados serializados
Flexibilidade ❌ Campos pré-definidos ✅ Qualquer texto pode ser variável
Agentes ❌ Difícil automação ✅ Perfeito para automação
Manutenção ❌ Dependências externas ✅ Sistema nativo
